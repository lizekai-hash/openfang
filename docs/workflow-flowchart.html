五种步骤模式一览
模式 行为 数据流
Sequential 上一步完成后顺序执行 current_input → prompt → output → 下一步的 current_input
FanOut 连续 FanOut 步骤并行执行 (join_all) 同一 current_input 分发给所有并行步骤
Collect 汇总所有 FanOut 输出 all_outputs.join("---") → current_input
Conditional 前一步输出包含 condition 才执行，否则跳过 条件满足时与 Sequential 相同
Loop 重复执行直到输出包含 until 或达到 max_iterations 每轮 output → 下轮 current_input
